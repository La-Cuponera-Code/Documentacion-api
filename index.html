    Registro de usuario

    Register

    Results

    {
  "code": "200",
  "message": "Email enviado correctamente"
}

    Api para registrar un usuario nuevo. Si viene el campo 'codigo_referido', se registra la relación de referido.

    HTTP Request

    POST http://localhost:9000/register

    Query Parameters

    Parameter Description
    email Email del usuario a registrar
    codigo_referido Codigo del usuario que lo refirió

    Verificación

    Verify

    Results

    {
  "code": "200",
  "message": "Usuario validado correctamente"
}

    Api para verificar el usuario mediante 2FA. Al verificarse, se elimina el temp_secret y se mueve a la tabla secret. Esto indica si fue validado o no.

    HTTP Request

    POST http://localhost:9000/verify

    Query Parameters

    Parameter Description
    email Email del usuario registrado
    token Token enviado por email

    Verify Login

    Results

    {
  "code": "200",
  "message": "Usuario validado correctamente"
}

    Api para verificar el login de un usuario mediante 2FA. El usuario debe estar validado. Si se valida correctamente, se guarda la sesión en la db.

    HTTP Request

    POST http://localhost:9000/verify_login

    Query Parameters

    Parameter Description
    email Email del usuario registrado
    token Token enviado por email

    Login

    Login

    Results

    {
  "code": "200",
  "message": "Email enviado correctamente"
}

    Api para login de un usuario existente. El usuario debe estar validado. Se manda el mail con el código generado a partir del secret.

    HTTP Request

    POST http://localhost:9000/login

    Query Parameters

    Parameter Description
    email Email del usuario registrado
